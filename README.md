## Usage

```
    Python 3
    Usage: python3 client.py server_port qname qtype timeout
    coding: utf-8

    Notes:
        Try to run the server first with the command:
            python3 server.py server_port
        Then run the client:
            python3 client.py server_port qname qtype timeout
```
